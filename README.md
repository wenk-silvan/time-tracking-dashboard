# Time tracking dashboard

This android app was built with the exercise from [Frontend Mentor](https://www.frontendmentor.io/challenges/time-tracking-dashboard-UIQ7167Jw).

## Launch application
Open the app in Android Studio and launch the application. 
You can either use a real android device or an android emulator. 
The minimal required SDK is 26. If you use a real device, 
make sure to enable Developer Mode and then USB Debugging before running the app.

## Technologies
- Android Studio Arctic Fox | 2020.3.1 Patch 3
- Kotlin
- Android MinSDK 26
- GSON
- Android Emulator Pixel 5 API 26
